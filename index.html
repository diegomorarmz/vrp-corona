Instrucciones Power BI
Paso 1 — Descarga los tres archivos .xlsx arriba.
Paso 2 — En Power BI Desktop: Inicio → Obtener datos → Excel → selecciona el archivo.
Paso 3 — Importa las tres tablas. Power BI detecta las columnas automáticamente.
Paso 4 — Relaciona por fecha (rutas ↔ optimización) y por prioridad (clientes ↔ rutas).
Paso 5 — Crea medida DAX: Ahorro = SUM(costo_actual) - SUM(costo_optimizado)
Tip — Visual de Línea para tendencia de costos, Mapa para clientes, Barras apiladas para antes/después.
